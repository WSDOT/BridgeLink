1 Managing BridgeLink Applications {#managing_applications}
2 ================================
3 BridgeLink is an extensible application framework. As you probably noticed during the installation process, you can choose to install any or all of the applications. BridgeLink provides a simple mechanism for managing the applications that were installed.
4 
5 ##Managing Project Types
6 BridgeLink Applications are presented to you as project types in the New Project window.
7 ![New Project Window](@ref NewProject.png)
8 
9 The available project types can be managed by enabling the ones you want to see and disabling the rest of them. Disabling a project type will remove it from the list of available project types in the New Project window but will not uninstall it.
10 
11 To manage project types
12 1. Select *File > Manage > Project Types*. This will open the Manager Project Types window
13 ![](ManageProjectTypes.png)
14 2. Check the boxes next to the project types that you want available
15 3. Press the [OK] button.
