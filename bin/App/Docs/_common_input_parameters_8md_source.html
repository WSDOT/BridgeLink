1 Common Input Parameters {#common_input_parameters}
2 =======================
3 Station, offset, bearing, and angle are common input parameters used in many of the BridgeLink appliccations. These parameters can be input with simple decimal values or in common engineering format.
4 
5 Notation
6 --------
7 The following notation is used to describe the common engineering format.

18 Stations
19 --------
20 Stations represent a position along an alignment. Stations can be input as a decimal number or as a number of full plus fractional chains.
21 
22 <table>
23 <tr><th>Format</th><th>Examples</th></tr>
24 <tr><td>[+|-]dddd.dd</td><td>233.32<br>-89.92</td></tr>
25 <tr><td>US Units<br>[+|-]dd+dd.dd</td><td>2+33.32<br>-0+89.92</td></tr>
26 <tr><td>SI Units<br>[+|-]dd+ddd.dd</td><td>2+343.32<br>-0+894.92</td></tr>
27 </table>

29 Offsets
30 -------
31 Offsets represent a perpendicular distance measured from a reference line. Left and right orientation is based on the observer looking in the positive direction of the reference line. If the reference line is an alignment, left and right are based on the observer looking ahead on station.
32 
33 <table>
34 <tr><th>Format</th><th>Examples</th></tr>
35 <tr><td>[+|-]dddd.dd<br>Values greater than zero are to the right of the reference line.</td><td>2343.32<br>-894.92</td></tr>
36 <tr><td>dddd.dd [L|R]</td><td>2343.32 R<br>894.92 L</td></tr>
