1 Download and Installation {#download_and_installation}
2 =========================
3 
4 ## Prerequisites
5 Before downloading BridgeLink make sure the following prerequisites are satisfied:
6 * Your computer has Microsoft Windows Version 7 or later
7 * The user account that will be installing the software has administrative privileges on the computer
8 
9  NOTE: Administrative privileges are not required to use BridgeLink
10 
11 ## Download
12 BridgeLink is available with a free download from the WSDOT web site
13 1. Start your favorite web browser and go to http://www.wsdot.wa.gov/eesc/bridge/software
14 2. Click on the BridgeLink Download Center
15 3. Press the [Download Now] button
16 4. Enter the requested information in the download form
17 5. Press the [Start Download] button
18 6. Press the [Save] button
19 
20 ## Create a PGSuper.com Account
21 If you haven't already done so, create an acount at www.pgsuper.com. This site, hosted by BridgeSight Inc., provides a peer-to-peer support forum, training materials, and an email newsletter.
22 
23 ## Install BridgeLink
24 BridgeLink is installed like all other Windows programs.
25 
26 ## Starting BridgeLink
27 BridgeLink is started like all other Windows programs. Select *Start > All Programs > Engineering Applications > BridgeLink > BridgeLink*.
28 
29 When BridgeLink starts you will be presented with the license agreement and disclaimer. It is a requirement of the Alternate Route Open Source License that these documents be presented everytime you start the program.
30 
31 The first time you run BridgeLink you will be prompted to configure PGSuper and PGSplice. The configuration procedure is the same for both applications. Enter your name and company in the User Information section. Press the [Update Configuration Now] button to download the default (WSDOT) configuration.
