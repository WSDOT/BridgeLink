Welcome to BridgeLink™

BridgeLink is an integrated environment that links together various bridge engineering software tools.

 

BridgeLink is developed by the Washington State Department of Transportation and is a member of the Alternate Route Project.

 

BridgeLink is licensed under the terms and conditions of the Alternate Route Open Source License. Please read this license carefully before using this software.

Getting Started

By itself, BridgeLink does little more than manage the bridge engineering software components that link into the BridgeLink environment. This document describes how to manage these software components. Refer to the documentation for each component to learn more about that component.

A Bit of History

For many years, WSDOT published PGSuper as a stand alone application. What was not obvious about this is that PGSuper, as is was known before BridgeLink was release, was in fact BridgeLink. At that time PGSuper and TOGA were the only applications to plug into the environment. Now that there are several plug-ins covering a variety of engineering capabilities, it doesn't make sense to refer to them as plug-ins to PGSuper.
